# Brain Tumor Detection using CNN
 
A deep learning project that detects brain tumors from MRI images using a Convolutional Neural Network (CNN).

## Features

- Brain tumor classification
- MRI image prediction
- Automatic tumour region highlighting with a red bounding box
- Data augmentation
- Early stopping
- Confusion matrix and classification report

## Dataset

Kaggle Dataset:
https://www.kaggle.com/datasets/navoneel/brain-mri-images-for-brain-tumor-detection

## Technologies Used

- Python
- TensorFlow/Keras
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn

## Accuracy

Approximately 90–96% test accuracy (may vary).

## How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Download the dataset from Kaggle.

3. Run the notebook:

```bash
jupyter notebook brain_tumor_detection.ipynb
```
